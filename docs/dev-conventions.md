# Development Conventions

## Branching
- main → Production
- develop → Integration
- devops → DevOps work
- ai-ml → AI/ML work
- data-analyst → Data work
- mern → MERN work

## Commit Messages
- feat: new feature
- fix: bug fix
- docs: documentation changes

## Pull Requests
- Create a PR when work is complete
- Request review before merging
